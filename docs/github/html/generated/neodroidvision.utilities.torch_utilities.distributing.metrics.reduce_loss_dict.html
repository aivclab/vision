neodroidvision.utilities.torch_utilities.distributing.metrics.reduce_loss_dict¶

neodroidvision.utilities.torch_utilities.distributing.metrics.reduce_loss_dict(loss_dict: dict) → dict[source]¶

Reduce the loss dictionary from all processes so that process with rank 0 has the averaged results. Returns a dict with the same fields as loss_dict, after reduction.
