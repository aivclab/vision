RandomFlipColor([ratio])

Randomly flips a colour channel in an image Each channel is individually evaluated against ratio (R) Hence, the chance of each channel (C) being affected is R^C and the chance of no channels being affected (N) is (1-R)^C and the chance of any channel(s) being affected is thus 1-N If you wish to calculate the R required for total risk T of any one or more channels being affected in a single forward, use this formula: R = 1 - (1 - T)^(1/C) For example, if your target T is 0.8 and there are 3 channels in an image, R = 1 - (1 - 0.8)^(1/3) = 0.415

RandomRGBNoise([ratio, val])

Inserts random R, G, and/or B values in an image Ratio affects the total amount of RGB values affected, not the amount of pixels.
