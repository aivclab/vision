neodroidvision.utilities.opencv_utilities.contour.analysis¶

Functions

find_extremes(c)

# determine the most extreme points along the contour

extent(cnt)

Extent is the ratio of contour area to bounding rectangle area.

solidity(cnt)

Solidity is the ratio of contour area to its convex hull area.

pixel_points(im_shape, cnt)

In some cases, we may need all the points which comprises that object.

point_contour_dist(cnt[, pnt])

In the function, third argument is " measureDist".

convexity_defects(cnt)

Notice that "returnPoints = False" in first line to get indices of the contour points, because input to convexityDefects() should be these indices, not original points.

equivalent_diameter(cnt)

Equivalent Diameter is the diameter of the circle whose area is same as the contour area.

orientation(cnt)

Orientation is the angle at which object is directed.
